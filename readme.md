## Installation via VS Code

1. Open the **Extensions** sidebar panel in VS Code.
2. Search for `Quite Canvas`
3. Click `Install`
4. Select either `Quiet Canvas — Dark` or `Quiet Canvas — Light` from the menu
5. Click `Reload`
